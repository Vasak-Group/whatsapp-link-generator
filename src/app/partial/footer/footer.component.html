Metodo de Uso

Area
Donde dice el area deberás poner el código de tu país. Si el teléfono es de Argentina tendrías que poner 54, para ver el código de tu país visita esta página.

Teléfono
Donde dice el teléfono deberás poner el número de teléfono al cual quieres que se le envie el mensaje cuando hagan click en el enlace, por ejemplo 1123882753.

Mensaje
Escribe el texto que aparecerá en el mensaje para enviar, una vez que el usuario haga click en el enlace. No se recomiendan mensajes largos pero si identificativos.
